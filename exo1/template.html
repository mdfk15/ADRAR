Nmap-usage

Description :

Une guide d'utilisation de base et avancé de nmap

Examples d'utilisation:

- nmap -F -t4 "host.ip.cible"/"CIDR" (Tester les ports les plus communs)

- nmap -O "host.ip.cible"/"CIDR" (Identifier le système)

- nmap -PA "n" "host.ip.cible"/"CIDR" (ping ACK)

- nmap -traceroute "host.ip.cible"/"CIDR" (Tracer la route pour les topologies)

- nmap -sT "host.ip.cible"/"CIDR" (Analiser un port en TCP)

- nmap -sV "host.ip.cible"/"CIDR" (Identifier la version d'un service)

- nmap -sC "host.ip.cible"/"CIDR" (Analiser avec les scripts plus communs
